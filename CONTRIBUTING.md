# Contributing to Awesome LLM Knowledge Bases

Thank you for your interest in contributing! This list aims to be a high-quality, curated collection of tools and resources for building LLM-powered knowledge bases.

## How to Add an Item

1. **Fork** this repository.
2. **Edit** `README.md` and add your item to the appropriate section.
3. **Submit** a pull request.

## Item Format

Each item should follow this format:

```
- [Name](https://url) - One line description that ends with a period.
```

- The description should be concise (one sentence).
- Start with a capital letter, end with a period.
- The description should explain what the tool does, not how great it is.
- Do not use promotional language like "best", "amazing", or "ultimate".

## Quality Standards

Before submitting, make sure the item:

- **Is relevant** to the LLM knowledge base workflow (ingest → compile → query → enhance).
- **Is actively maintained** — the project should have commits within the last 12 months.
- **Has a working URL** — broken links will not be accepted.
- **Is not a duplicate** — check that the item isn't already in the list.
- **Has meaningful adoption** — projects should have at least 50 GitHub stars or equivalent community traction.

## Suggesting a New Section

If your item doesn't fit any existing section, you can propose a new one. Please explain in the PR description why the new section is needed.

## Reporting Issues

Found a broken link or outdated entry? Please [open an issue](../../issues/new) with:

- The item name and section
- What's wrong (broken link, project archived, etc.)
- A suggested replacement, if applicable

## PR Checklist

- [ ] Item follows the correct format: `- [Name](url) - Description.`
- [ ] URL is valid and points to the correct project
- [ ] Description is concise and objective
- [ ] Item is added in alphabetical order within its section
- [ ] No trailing whitespace or formatting issues
- [ ] PR title follows the format: `Add [Item Name]`

## Code of Conduct

Be respectful and constructive. This is a community resource — let's keep it welcoming for everyone.
